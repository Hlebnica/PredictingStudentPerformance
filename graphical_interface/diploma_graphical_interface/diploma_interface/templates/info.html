Информация о программе

Данная программа является реализацией дипломной работы на тему "Разработка системы для предсказания успеваемости студентов на основе данных портала «Электронный ЮУрГУ»"
Научный руководитель: доктор физ.-мат. наук М.Л. Цымблер
Автор работы: студенты группы КЭ-403 Д.В. Старостенок

Система предоставляет функционал:

  • Извлечение данных из:
    • API системы "Универис"
      • Для получения данных необходимо указать:
      • Диапазон годов
      • Диапазон учебных курсов (от 1 до 4)
      • Высшую школу
    • Сохраненных CSV файлов
      • Для получения данных необходимо загрузить следующие CSV файлы:
      • Журнал за год
      • Информация о студентах
      • Рейтинг студентов по предметам
      • Результаты ЕГЭ студентов
  • Анализ данных
    • Для работы необходимо выполнить извлечение данных любым указанным выше способом
    Далее необходимо указать:
    • Модель данных
    • Предмет или Тип контроля предмета в зависимости от модели
    • Аналитическую модель
    • Гиперпараметры
